Australia's Economic Heartbeat

Australia's Economic Heartbeat was created for GovHack 2015 by The Tell-Tale Hearts.

Source code available on GitHub.

Data Sources

Aggregate indicator data is calculated using data from the following sources:

  • ABS.Stat
  • Australian Prudential Regulator
  • ACNC for Charity count data
  • IPGOD for patent application data
